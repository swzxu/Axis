# ![Axis](https://raw.githubusercontent.com/swzxu/Axis/refs/heads/main/banner.png)

Axis is a simple Flutter proxy client powered by Mihomo.

![Home page](https://raw.githubusercontent.com/swzxu/axis/refs/heads/main/Axis%202_b1.png)
## Features
* Subscription support
* DNS settings
* Material 3 with white and dark theme
* Autorun on boot
* Color theme customization
## Run

```bash
flutter pub get
flutter run -d windows
```

## Build

```bash
flutter build windows --release
```
